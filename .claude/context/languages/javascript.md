# JavaScript Standards & Guidelines

## Project Structure & Management
- ALWAYS produce self-contained, vanilla JavaScript code... with few/no dependencies.
- Use `npm` / `npx` if neccessary.


## Guidelines
- NEVER use JavaScript for server-side code. JavaScript should ONLY be used for browser/client-side code.
- Prefer async/await over promise chains
- Use functional patterns where appropriate
- Handle errors at appropriate boundaries
- Avoid callback hell with modern patterns
- Consider bundle size for browser code


## Preferred Technologies, Libraries & Frameworks
- **AG Grid**: For creating data tables/grids.
- **AG Charts**: For creating charts & plots.
- **Mousetrap**: For handling keyboard shortcuts.
